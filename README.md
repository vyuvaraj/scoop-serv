# Scoop Bucket for Serv

[Serv](https://github.com/vyuvaraj/Serv-lang) is a programming language for building backend services, schedulers, and APIs. It compiles to native binaries via Go.

## Install

```powershell
scoop bucket add serv https://github.com/vyuvaraj/scoop-serv
scoop install serv
```

## Update

```powershell
scoop update serv
```

## Verify

```powershell
serv --help
```

## Requirements

- [Go 1.22+](https://go.dev/dl/) must be installed
